# OQBI-Redesign

This project is a front-end website prototype for OQ Base Industries built using HTML, CSS, and JavaScript. HTML is used to create the structure of the website, such as the sections, navigation bar, and content areas. CSS is used to design the visual appearance, including colors, layout, animations, and responsive design so the website works well on desktop and mobile devices. JavaScript adds interactivity, such as switching between pages without reloading, opening pop-up windows (modals) for additional information, and handling language switching between English and Arabic with right-to-left support for Arabic. The site is organized using reusable components like cards, carousels, and navigation menus, which makes the design consistent and easier to maintain or expand in the future. Overall, the structure follows modern web design practices to create a visually engaging, responsive, and bilingual corporate website interface.
